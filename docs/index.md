# Global Design System Documentation

Welcome to the GDS documentation. This design system provides consistent, accessible UI components for building applications.

## Components

### Atoms
- Basic building blocks: buttons, icons, inputs, labels

### Molecules
- Composite components: search bars, form fields, cards

### Organisms
- Complex sections: navigation, headers, footers
