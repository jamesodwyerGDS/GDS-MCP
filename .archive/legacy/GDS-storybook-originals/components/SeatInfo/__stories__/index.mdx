## Usage

Props include:

`level`, `section`, `row` and `seat` objects. Each must contain a `title` (string) and `details` (string). If `row.details` has not been passed in, then only the section information is displayed. If `seat.details` has not been passed in, then the seat section displays "-".

<Canvas of={stories.NoLevel} />
<Canvas of={stories.NoSeatDetails} />
<Canvas of={stories.NoRowDetails} />

`timedEntryDetails` (string, optional) - if this has been filled in, the component will just display this text in bold (ignoring any `level`, `section`, `row` and `seat` props passed in):

<Canvas of={stories.TimedEntry} />

`isDisplayInline` (boolean, optional , `false` by default). If this is `true`, then the information is displayed in a single row and uncoloured.

<Canvas of={stories.DisplayInline} />
<Canvas of={stories.DisplayInlineNoRow} />

`description` (string, optional) - displays a description below the seat info block

<Canvas of={stories.WithDescription} />

`styleVariant` (string, optional, `default` by default) - displays a new variant with updated text style and without line dividers when set to `noDivider`

<Canvas of={stories.NoDivider} />
