## Rendering

Each SidePanel component is rendered as a portal to a wrapper element with the Id **{GdsPortalRootId}**, which is appended directly onto **document.body**.
This setup isolates the SidePanel from the rest of the elements on the document to provide a consistent position for styling and managing z-index.
